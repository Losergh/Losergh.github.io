ZFC-Axiom

Posted on 2025-08-24

Unless otherwise stated, all articles are based on the ZFC axiomatic system, namely:

(i) Axiom of the Empty Set: The empty set exists.

(ii) Axiom of Extensionality: If two sets have the same elements, then the two sets are equal.

(iii) Axiom of Separation: Let \(P(a)\) be a formula. For any set \(A\), the set \(B=\{a \mid a \in A,\; P(a)\ \text{holds}\}\) is also a set.

(iv) Axiom of Pairing: For any sets \(A, B\), there exists a set whose elements are exactly \(A\) and \(B\). In particular, for any set \(A\), the set \(\{A\}\) also exists.

(v) Axiom of Union: If \(\mathcal{X}\) is a set of sets, then there exists \(\underset {X\in\mathcal{X}}{\cup}X\), whose elements are precisely all elements of the members of \(\mathcal{X}\).

(vi) Axiom of Power Set: For any set \(X\), there exists a set \(P(X)\) consisting of all subsets of \(X\) (called the power set of \(X\)).

(vii) Axiom of Infinity: There exists an infinite set (definition: a set that is not finite).

(viii) Axiom of Replacement: Let \(P(x,y)\) be a formula such that for every \(x\) there exists a unique \(y\) with \(P(x,y)\) holding. Then for any set \(X\), there exists a set \(Y=\{y \mid P(x,y)\ \text{holds for some } x \in X\}\).

(ix) Axiom of Regularity (Foundation): Every nonempty set has a minimal element under the membership relation, i.e., there exists an element \(a\) of \(X\) such that for every \(b \in X\) with \(b \neq a\), we have \(b \notin a\).

(x) Axiom of Choice (AC): If \(X\) is a set consisting of nonempty sets as its elements, then there exists a set that contains exactly one element from each member of \(X\).
